# Bay Ave Barnacle — Flood Prediction Model v0.10

**Promotion date:** 2026-07-09, the night of EVENT #4 (the 5.53 in/hr
flash flood ON the high tide — top-3 all-time), fitted while the
recession data was hours old.

**One change from v0.9, and it's the big one: the pluvial pathway is
now a DYNAMIC TANK MODEL** — the water_series rain line is a true
predicted hydrograph (rise, peak, recession, with timing), closing
the "pluvial timing is unmodeled" gap open since 7/6.

Everything else — tide-keyed level path (enhancement 0.00, −2.82
datum, wind adjustment −0.13 offshore), 18 landmarks, stage-storage
curve, head-dependent drainage, NWS alerts ingestion, banner
scenario brackets — carries over from v0.9 unchanged
(`model/archive/v0.9.md` for full detail; still authoritative for
those parts).

## The tank

```
dV/dt  = TANK_K · max(0, R(t − lag) − D(bay))^TANK_GAMMA  −  TANK_KOUT · V
stage  = stage_storage_curve(fill from tide-set base, +V)
water  = 3.52 + stage/12          (NAVD88)

TANK_K     = 1.265e6 cell-in/h at 1 in/hr net    (fitted)
TANK_GAMMA = 0.70                                 (fitted)
TANK_KOUT  = 3.12 /h  (~19-min pool e-folding)    (fitted)
TANK_LAG   = 14 min   (hillside concentration)    (fitted; observed 16–20)
D(bay)     = head-dependent drain (0.25 in/hr below bay 3.0 → 0 at 3.52)
```

Inflow: intensity-dependent delivery of catchment-amplified rain.
Outflow: linear reservoir (drains + road conveyance off the shelf) —
calibrated by the two measured recessions. R input: raw hourly QPF
bucket rates (the tank does its own lag/smoothing; the old two-hour
pre-smoothing is retired).

## Honesty notes

1. γ = 0.70 in TRUE-rate space with duration explicit. The
   2026-07-07 "C grows with intensity" finding was partly
   duration-confounded; with T explicit the response is mildly
   sub-linear. Michaelis–Menten remains unnecessary.
2. Input remains the weak link: QPF smears/misses convective cells
   (7/9 forecast burst 0.74 vs actual 5.53). Tank physics validated
   ≠ forecast skill; alerts + analog floors carry the gap until
   MRMS nowcasting.
3. k_out is a single linear constant covering drains + road
   conveyance; stage-dependence untested above +19″.
4. Connected-pool + curve lower-bound caveats carry from v0.9.

## What might break this model

1. A long-duration moderate event (tests k_out/γ trade-off far from
   the burst regime).
2. Bay ABOVE grate tops during a burst (Oct-30-class compound at
   +2 ft tide) — tank base handling verified only to base = grates.
3. QPF input failure modes (see honesty note 2).
